Spring Bootでのデータベース接続について質問があります
内容

Spring Bootアプリケーションで、MySQLデータベースに接続する際に問題が発生しています。

application.propertiesファイルに以下のように設定しましたが、起動時に接続エラーが表示されます：

spring.datasource.url=jdbc:mysql://localhost:3306/mydb
spring.datasource.username=root
spring.datasource.password=password
spring.datasource.driver-class-name=com.mysql.cj.jdbc.Driver
spring.jpa.hibernate.ddl-auto=update

エラーメッセージは以下の通りです：

java.net.ConnectException: Connection refused: connect

MySQLサーバーは起動していることを確認済みです。どのように解決すればよいでしょうか？

鈴木一郎 2025/01/15 15:45

ポート番号が正しく設定されているか確認してみてください。デフォルトのMySQLポートは3306ですが、別のポートを使用している可能性があります。

佐藤花子 2025/01/15 16:20

MySQLのユーザー権限も確認してみてください。rootユーザーで接続しようとしていますが、適切な権限が設定されていない可能性があります。
